<Steps>
<Step title="Create an endpoint">

Sign in and click **New Endpoint** in the dashboard. You get a unique URL like:

```
https://go.webhooks.cc/w/<slug>
```

Or create one programmatically with the [SDK](/docs/sdk) or [CLI](/docs/cli).

</Step>
<Step title="Send a webhook">

Point your service at the endpoint URL. Or test it with curl:

```bash
curl -X POST https://go.webhooks.cc/w/<slug> \
  -H "Content-Type: application/json" \
  -d '{"event": "payment.success", "amount": 4999}'
```

The receiver accepts any HTTP method, content type, and body.

</Step>
<Step title="View in the dashboard">

Open the [dashboard](/dashboard). Requests appear in real-time. Inspect headers, body, query parameters, and metadata. Copy as curl, replay to another URL, or export as JSON/CSV.

</Step>
</Steps>
