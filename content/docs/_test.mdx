## API Method

<ApiMethod method="POST" path="/endpoints" title="Create Endpoint">

Creates a new webhook endpoint and returns its URL and slug.

<ParamTable>

| Param       | Type      | Required | Description                   |
| ----------- | --------- | -------- | ----------------------------- |
| `name`      | `string`  | no       | Display name for the endpoint |
| `ephemeral` | `boolean` | no       | Auto-delete after TTL         |

</ParamTable>
</ApiMethod>
